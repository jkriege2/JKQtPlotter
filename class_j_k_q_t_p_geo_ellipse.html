Detailed Description

This JKQTPGeometricPlotElement is used to draw an ellipse.

Note
This function support the JKQTPGeometricPlotElement::DrawMode::DrawAsMathematicalCurve only. This means that ellipses are always treated as mathematical curves, as no meaningful parametrization for ellipses on non-linear axes could be found! In the mode JKQTPGeometricPlotElement::DrawMode::DrawAsGraphicElement the points are connected by straight lines, independent of the linearity or non-linearity of the coordinate axes.
See also
http://www.codeguru.com/cpp/g-m/gdi/article.php/c131 and http://en.wikipedia.org/wiki/Ellipse#General_parametric_form
Example (JKQTPlotter): Plotting Geometric Objects, JKQTPGeoPie, JKQTPGeoChord, JKQTPGeoArc
