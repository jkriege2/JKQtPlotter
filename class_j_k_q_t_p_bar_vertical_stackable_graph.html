Detailed Description

This implements a vertical bar graph with bars between and Optionally several graphs of this type may be stacked on top of each other.

Draw stacked barcharts by connecting several plots by calling setStackedParent(belowPlot) for each plot

See also
JKQTPBarVerticalGraph, Example (JKQTPlotter): Simple stacked barchart
