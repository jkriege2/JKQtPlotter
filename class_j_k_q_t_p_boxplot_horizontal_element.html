Detailed Description

This implements a horizontal (notched) boxplot where the data is directly given to the object and not stored in a column, as in JKQTPBoxplotVerticalGraph.

the x position is given in pos. All other data are given in the median, min, max, percentile25 and percentile75.

Note
See JKQTPBoxplotVerticalElement for a detailed documentation
